# CUMTB 毕业论文 LaTeX 精简版模版

这是一个专为中国矿业大学（北京）学生优化的、开箱即用的 LaTeX 毕业论文模版。本模版已经过深度整理和精简，将复杂的类文件逻辑转化为了更加直观、易用的“分块式”源码结构。

## 📁 目录结构

为了您的写作专注，文件夹已按功能模块化：

*   **`data/` (核心工作区)**: 📚 **这里是您写作的唯一入口。**
    *   `main.tex`: 总控文件，配置个人信息。
    *   `6Body.tex`: 论文正文、致谢、参考文献、附录都在这里。
    *   `main.pdf`: 当前生成的示例文档（包含一份详尽的 LaTeX 使用教程）。
*   **`figures/` (素材区)**: 🖼️ 存放所有插图文件。
*   **`docs_and_standards/` (参考库)**: 📖 包含学校官方的 Word 规范、封皮范例及项目文档。
*   **`archives/` (归档区)**: 📦 存放原始的系统级类文件（.cls）和旧版源码。

## 🚀 快速开始

### 1. 修改个人信息
打开 `data/main.tex`，在文件开头的配置区域修改论文题目、姓名、学院等信息。

### 2. 撰写正文
打开 `data/6Body.tex`。模版中已经为您写好了一篇**《LaTeX 学术排版指南》**，您可以直接参考其中的代码来学习如何插入：
*   **数学公式**（麦克斯韦方程组对齐示例）
*   **高质量图表**（三线表与合并单元格示例）
*   **多子图排版**（左右并排对比示例）
*   **自动化引用**（GB/T 7714 标准引用示例）

### 3. 编译生成
推荐使用编辑器（如 VS Code + LaTeX Workshop 或 TeXstudio）直接点击编译，或者在终端运行：
```bash
cd data
xelatex main.tex
xelatex main.tex
```
*(注意：必须使用 **XeLaTeX** 引擎编译，且至少编译 2-3 次以同步引用标号)*

## ✨ 核心特性

- **简单直观**：无需学习复杂的 `.cls` 宏定义，所有样式都通过标准 `data/main.tex` 中的宏包控制。
- **自动对齐**：致谢段落、参考文献对齐、数学公式对齐均已调优。
- **真实教程**：模版内容本身就是一份论文案例，涵盖了毕业设计 90% 的排版场景。

## ⚠️ 常见问题 (FAQ)

*   **引用显示 [?]**：请连续点击编译按钮 3 次。
*   **字体报错**：确保系统中安装了：宋体、黑体、楷体、Times New Roman。
*   **公式出错**：请检查公式环境是否完整闭合。
