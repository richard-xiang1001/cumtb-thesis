# 更新日志

本文档记录 CUMTB Thesis LaTeX 模板的版本更新历史。

## [2.0.0] - 2025-12-05

### 新增 ✨
- **Makefile 构建系统**：提供自动化编译命令（thesis, clean, help等）
- **latexmk 配置**：支持 `latexmk -xelatex` 一键编译
- **盲审模式**：新增 `review` 选项，自动隐藏作者和导师信息
- **原创性声明和授权书**：添加 `data/statement.tex`，符合学位论文规范
- **符号说明环境**：完整实现基于 longtable 的符号表
- **GitHub Actions CI/CD**：自动编译测试和PDF构建
- **使用手册**：详细的 `docs/cumtbthesis.tex` 用户文档
- **贡献指南**：添加 `CONTRIBUTING.md`
- **字数统计脚本**：`scripts/wordcount.sh` 统计论文字数和图表数量

### 改进 🔧
- 优化符号说明环境，支持跨页显示
- 更新 README.md，添加新功能使用说明
- 盲审模式下自动隐藏原创性声明

---

## [1.0.0] - 2025-12-05

### 新增 ✨
- 初始版本发布
- 基于 ctexbook 的文档类
- 支持本科、硕士、博士三种学位类型
- 使用 biblatex + biber 管理参考文献（GB/T 7714-2015）
- 封面、摘要、目录、正文、参考文献、致谢等完整结构
- 图表、公式、代码、算法环境
- 详细的 README.md 和 INSTALL.md

### 文件
- `cumtbthesis.cls`：文档类文件
- `cumtbthesis.cfg`：配置文件
- `main.tex`：主文档示例
- 示例章节：chap01.tex, chap02.tex, chap03.tex
- 参考文献示例：refs.bib

## 版本命名规范

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 2.0.0 规范：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

## 链接格式

- `新增` - 新功能
- `改进` - 现有功能的改进
- `修复` - Bug 修复
- `文档` - 文档相关
- `性能` - 性能优化
- `重构` - 代码重构
- `测试` - 测试相关
- `构建` - 构建系统或依赖项
- `移除` - 移除的功能或文件
