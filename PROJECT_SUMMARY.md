## 特色功能

### 1. 智能盲审模式
```latex
\documentclass[master,review]{cumtbthesis}
```
- 自动隐藏作者、导师信息
- 隐藏原创性声明
- 隐藏致谢部分

### 2. 完整附录支持
```latex
\appendix
\chapter{补充材料}
```
- 自动编号：附录A、附录B
- 图表编号：A-1, A-2

### 3. 现代子图环境
- 使用 subcaption 宏包
- 支持灵活布局
- 自动子图标签

### 4. 自动化构建
```bash
make thesis    # 一键编译
make clean     # 清理文件
make wordcount # 统计字数
```

## 使用流程

### 1. 获取模板
```bash
git clone https://github.com/richard-xiang1001/cumtb-thesis.git
cd cumtb-thesis
```

### 2. 配置信息
编辑 `main.tex` 填写论文信息

### 3. 编译论文
```bash
make thesis
```

### 4. 查看结果
打开生成的 `main.pdf`

## 技术特点

- **编译引擎**：XeLaTeX
- **中文支持**：ctex + xeCJK
- **参考文献**：biblatex + biber (GB/T 7714-2015)
- **文档类**：基于 ctexbook
- **版本控制**：Git
- **CI/CD**：GitHub Actions
